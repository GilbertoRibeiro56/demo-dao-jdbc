Este projeto é uma aplicação demo em Java que tem como objetivo demonstrar, de forma prática e didática, o uso de JDBC em conjunto com o padrão de projeto DAO (Data Access Object) para acesso e manipulação de dados em um banco de dados relacional.

O foco do projeto é separar a lógica de persistência da lógica de negócio, seguindo boas práticas de arquitetura de software.

Objetivo do Projeto

Demonstrar o uso do JDBC para conexão com banco de dados
Aplicar o padrão DAO (Data Access Object)
Facilitar o entendimento da camada de persistência
Servir como base de estudo ou projeto inicial para aplicações Java com banco de dados


Como Executar o Projeto
Clone o repositório:
git clone https://github.com/seu-usuario/seu-repositorio.git

Crie um banco de dados no SGBD de sua preferência
Configure as credenciais no arquivo de conexão (db.properties)



Este projeto é apenas para fins educacionais e de demonstração.
